Hello Phaser

Welcome to your first game development project with Phaser 3

Introduction to Phaser

Phaser is a fast, free, and fun open source HTML5 game framework that offers WebGL and Canvas rendering across desktop and mobile web browsers. Games can be compiled to iOS, Android and native apps using 3rd party tools.

In this assignment, you'll build a simple interactive game from scratch using JavaScript and Phaser 3. You'll learn how to set up a project, create game files, implement interactivity, and deploy your game online.

What You'll Learn

By completing this assignment, you'll learn:

  • How to set up a Phaser 3 project with Vite
  • Creating and structuring game files from scratch
  • Loading and displaying game assets (images and sounds)
  • Creating interactive game objects
  • Handling user input (mouse/touch)
  • Managing game state (score tracking)
  • Deploying your game to Netlify
Start the Assignment

Project Structure

Unlike tutorials that provide pre-written code, this assignment will have you create key files yourself. Your starter code includes:

  • Project configuration (package.json with Phaser dependency)
  • Build tools (Vite for modern JavaScript development)
  • Deployment configuration (netlify.toml)
  • Game assets (images and sounds)

You'll be building the main game files (main.js and MainScene.js) yourself, giving you a deeper understanding of how Phaser games are structured.

Why Game Development?

Game development is not only fun, but it's also an excellent way to practice programming concepts in a visual and interactive context. Through game creation, you'll strengthen your understanding of:

  • Object-oriented programming
  • Event-driven architecture
  • Real-time processing
  • User experience design
  • Asset management
  • Modern web deployment workflows

These skills are valuable across many programming domains, not just in game development.
